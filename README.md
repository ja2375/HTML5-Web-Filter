Web-Filter-beta
===============

Web application for applying Instagram-like effects on photos

Effects available:

  - Clarity
  - Concentrate
  - CrossProcess
  - GlowingSun
  - Grungy
  - HazyDays
  - Hemingway
  - HerMajesty
  - Jarques
  - Lomo
  - Love
  - Normal
  - Nostalgia
  - OldBoot
  - OrangePeel
  - Pinhole
  - SinCity
  - Sunrise
  - Vintage
